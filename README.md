# MoneyMetric

MoneyMetric is an Android application built with Jetpack Compose that helps you track your finances.

## Features

*   Track your income and expenses.
*   Categorize your transactions.
*   View your financial history.

## Getting Started

To get started with MoneyMetric, you'll need to have Android Studio installed.

1.  Clone the repository: `git clone https://github.com/your-username/moneymetric.git`
2.  Open the project in Android Studio.
3.  Run the app on an emulator or a physical device.
